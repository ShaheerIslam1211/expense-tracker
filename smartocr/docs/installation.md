# Installation

## Stable release

To install ocr_image_to_text, run this command in your terminal:

```sh
uv add smartocr
```

Or if you prefer to use `pip`:

```sh
pip install smartocr
```

## From source

The source files for ocr_image_to_text can be downloaded from the [Github repo](https://github.com/ShaheerIslam1211/smartocr).

You can either clone the public repository:

```sh
git clone https://github.com/ShaheerIslam1211/smartocr
```

Or download the [tarball](https://github.com/ShaheerIslam1211/smartocr/tarball/main):

```sh
curl -OJL https://github.com/ShaheerIslam1211/smartocr/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd smartocr
uv sync
```
